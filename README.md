Лаунчер для запуска майнкрафт

Дипломный проект для автоматизации, упрощения работы с модифицированными сборками и данными.

Программа включает в себя окна для авторизации, администратора и клиента.

Окно для администратора включает в себя 6 модулей:
- главная страница (небольшая сводка данных)
- подписки (управление и просмотр)
- сборки (управление и просмотр)
- новости (управление и просмотр)
- клиенты (просмотр)         
- служба поддержки (просмотр с возможностью удаления)
Окно для клиента включает в себя 7 модулей:
- профиль (управление информацией, аккаунтом и подписками)
- главная страницы (выбор сборки и ее последующий запуск)
- сборки (просмотр и скачивание сборок)
- новости (просмотр с возможностью лайкнуть или дизлайкнуть запись)
- друзья (просмотр, поиск, удаление и добавление друзей)
- служба поддержки (отправка сообщений с возможностью прикрепления фото разработчикам)
- настройки (управление модификациями и параметрами запуска игры)

Технологии:
- язык программирования C# 12.0
- фреймворк Visual Studio 2022
- база данных MySQL|OpenServer 5.2.2
- фронтенд XAML|WPF
- дополнительно Git

Особенность проекта:
Проект разработан как локальное решение для демонстрации возможностей в автономном режиме.
