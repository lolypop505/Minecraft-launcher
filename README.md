Лаунчер для запуска майнкрафт

Дипломный проект для автоматизирования, упрощения работы с модифицированными сборками и данными.

Основные алгоритмы, реализованные в системе:
- Регистрация и авторизация. Пользователь вводит логин, пароль и почту. Данные проверяются на корректность и уникальность. После успешной регистрации информация сохраняется в базе данных. При авторизации происходит сверка введённых данных с сохранёнными.
- Профиль пользователя. Пользователь может изменить свои данные, сменить пароль или удалить аккаунт. При каждом действии происходит проверка данных на корректность и обновление информации в базе данных.
- Лента новостей. При запуске лаунчера загружается список новостей, добавленных администратором через панель управления. Пользователи могут ставить лайки и дизлайки, которые сохраняются и учитываются при отображении записей.
- Сборки. Пользователю отображаются доступные сборки. Алгоритм проверяет наличие подписки, после чего предоставляется доступ к загрузке файлов.
- Подписки. Подписка оформляется через страницу профиля. После покупки в базе данных сохраняется информация о сроке действия и уровне подписки. 
- Техническая поддержка. Пользователь может отправить сообщение об ошибке через специальную форму. Сообщение сохраняется в базе данных и становится доступно администратору для просмотра и обработки.
- Система друзей. Пользователь может найти других игроков по нику и отправить заявку в друзья, после чего данная заявка отобразится у получателя. Все действия с пользователями сохраняются в соответствующие таблицы базы данных.
- Настройки. Пользователь изменить разрешение, выделяемую память, директорию, режим отображения окна, а также включать и выключать модификации в зависимости от выбранной папки. 
- Окно администратора. Администратор может добавлять, редактировать и удалять новости, сборки и подписки, а также просматривать списоки пользователей, сообщений из тех. поддержки и общую сводку данных.

Технологии:
- Язык программирования C# 12.0
- Фреймворк Visual Studio 2022
- База данных MySQL (OpenServer 5.2.2 локальная сборка)
- Фронтенд XAML|WPF
- Дополнительно Git

Особенность проекта:
Проект разработан как локальное решение для демонстрации возможностей в автономном режиме.
